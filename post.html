Network Automation Configuration Management

This was a project I started back in november 2019 when I first learned python. I was the sole creator/developer for the first year of the project before I was able to hire help.

The system stored fuel sample data for the entire Cisco & Mikrotik router

features :

Routing: The basic configuration of static, dynamic OSPF, RIPv1, RIPv2, and BGP routing.
Vlan: Configure vlan id
Code Based: Send text based configuration to the destination device
Backup: Backup device configuration
Restore: Restore the device configuration
Settings: Add and delete vendor configurations

Technologies:
  • - Python
  • - Django + Django REST Framework
  • - JavaScript
  • - SQLite
  • - Ubuntu Server
  • - GNS3
  • - Router Cisco & Mikrotik
